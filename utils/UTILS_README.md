# Utilities and Helpers

Helper functions and utility scripts for competitive programming.

## Included Utilities

### Testing Utilities

- **Input validators**: Check input format compliance
- **Output checkers**: Verify output matches expected format
- **Test generators**: Create random test cases
- **Performance profilers**: Measure time/space usage

### Common Algorithms (Quick Reference)

- **Math utilities**: GCD, LCM, prime checking
- **Array helpers**: Sorting, searching, transformation
- **String utilities**: Pattern matching, substring operations
- **Graph helpers**: Path finding, connectivity

### Development Tools

- **Template starter**: Boilerplate for competitive programming
- **Debug helpers**: Print statements, asserts
- **Benchmarking**: Compare algorithm performance
- **Solution checker**: Validate against test cases

## Usage

Import utilities as needed:

```python
from utils import fast_input, read_ints, gcd
```

## Adding New Utilities

1. Create module in utils directory
2. Add clear docstrings
3. Include usage examples
4. Test thoroughly
5. Update this README
